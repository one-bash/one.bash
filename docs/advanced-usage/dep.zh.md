# ONE Dependencies

依赖：

- [dotbot][]: 通过配置创建符号链接并管理它们。
- [composure][]: 提供模块用的基础函数。

你可以使用 `one dep` 命令管理依赖。

- `one dep install` 安装所有依赖到 [deps/](../../deps).
- `one dep update` to update all dependencies.
- `one dep update <dep>` to update a dependency.
